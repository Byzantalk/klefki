In number theory, an integer q is called a quadratic residue modulo n if it is congruent to a perfect square modulo n; i.e., if there exists an integer x such that:

\[x^{2}\equiv q{\pmod {n}}\]

Otherwise, q is called a quadratic nonresidue modulo n.

The Legendre symbol¶

\[\begin{split}\left(\frac{a}{p}\right) = \left\{ \begin{aligned} &0\ if \ p \mid a \\ &1\ if\ p \nmid a\ \text{and a is a quadratic residue mod p} \\ &-1\ \text{if a is quadratic none-residue mod p} \end{aligned} \right.\end{split}\]
